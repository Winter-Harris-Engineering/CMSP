# Cyber Mission Support Platform (CMSP)
This repository contains the promotional front-end application for the Cyber Mission Support Platform (CMSP). It is designed to showcase the platform's capabilities in accelerating Risk Management Framework (RMF) automation and NIST 800-53 compliance. The site utilizes a strict Enterprise SaaS dark-mode aesthetic to present features such as compliance velocity metrics and AI-driven evidence analysis.

Tech Stack

* React
* Vite
* Tailwind CSS
* Shadcn UI (with Recharts)
* React Router

Local Development
To interact with the development server, ensure you have Node.js installed on your machine. Run the following commands from the root directory of the project:

Install the dependencies:

```Bash
npm install
```

Start the Vite development server:

```Bash
npm run dev
```

View the application:

Open your web browser and navigate to the local address outputted in your terminal window (typically http://localhost:5173). The server features hot-module replacement (HMR), meaning any saved changes to the React components will automatically reflect in the browser.
